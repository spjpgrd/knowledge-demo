# Inbox

Drop unstructured content here — links, voice transcripts, screenshots, brain dumps, anything.

Classify and file into `library/` or `work/` when you're ready.

If using Claude Code (or another AI assistant that can read local files):

- `/ingest-articles <url>` — pull articles from the web straight into `library/` (or here, if classification is uncertain). See `examples/workflows/ingest-articles.md`.
- `/tidy-knowledge` — process whatever is sitting in this inbox and file it under the right Dewey code. See `examples/workflows/tidy-knowledge.md`.
