# Dewey Decimal Codes — Knowledge Library

Authoritative reference for category names. ALWAYS consult this file before creating directories in `library/`.

## Naming Convention

- **Range folders:** `{hundred}-{lowercase-hyphenated-name}` (e.g., `100-philosophy-and-psychology/`)
- **Category folders:** `{code}-{lowercase-hyphenated-name}` (e.g., `150-psychology/`)
- Categories nest under their range parent: `library/600-technology/658.8-marketing-management/`
- To target a broad area, reference the range folder (e.g., "look at the `100` section")
